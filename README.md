# Explicación de la Tarea :bookmark_tabs:
El objetivo de esta tarea es realizar un análisis de datos sobre el dataset ny_citibikes_raw.xlsx, el cual contiene información de viajes realizados con bicicletas públicas en la ciudad de Nueva York.
El análisis incluye:

- Carga y limpieza del dataset
- Cálculo de estadísticas 
- Creación de nuevas columnas
- Identificación de patrones de uso
- Visualización de tendencias del usuario
- Interpretar el comportamiento de los usuarios

El propósito es entender cómo se utilizan las bicicletas compartidas a lo largo del tiempo y detectar diferencias entre tipos de usuarios, estaciones, días de la semana y otros factores.
Tambien buscaremos errores o datos fuera de lugar.
